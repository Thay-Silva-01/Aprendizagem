Formatação e ênfase

A tag b é usada para chamar a atenção para um trecho de texto sem indicar que ele tem importância especial. Deve ser usada apenas quando nenhuma outra tag semântica for apropriada.
title="Negrito sem semântica!"

A tag strong dá ao texto um forte sentido de importância. Visualmente é renderizado em negrito, mas semanticamente transmite que o conteúdo é significativo.
title="Negrito com semântica!"

A tag < em > indica ênfase textual. Visualmente é renderizado em itálico, mas semanticamente transmite que o conteúdo deve ser enfatizado.

A tag < mark > é usada para destacar partes do texto que são relevantes no contexto atual, como se tivessem sido marcadas com um marcador amarelo.
